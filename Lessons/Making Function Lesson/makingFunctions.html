Lesson 5: Making Functions

Functions or methods group together a sequence of statements that perform a task. A function is defined inside a class and should have a descriptive name that represents what the function does.

Here is an example of a function:
public int getSum (int parameter, int parameter2){
int sum = value + value2;
return sum;
}
Notice that the function returns an integer and there are two parameters that the function takes in. Each function has a method signature which includes the access modifier that determines the scope to which the method can be accessed, return type, function name, and parameters. The signature in this example is: public int sampleFunction (int parameter, int parameter2). Ensure you return the correct data type as defined in your method signature.
