Lesson 5: Making Functions

A function can return many other types of data such as a String, an array, and more. A function can also return nothing.

Take a look at the example below:
public void swapValue (int parameter, int parameter2){
int temp = parameter;
parameter = parameter2;
parameter2 = temp;
}
The function swaps the values of the parameters. The return type here is void. There is no return statement because a void a function doesn't return anything. A void function may be useful in a situation such as printing out a value.
